# Sudoku game developed on HoloLens 2
## Current feature
- Core sudoku gameplay (validate the number filled in by player)
- Undo action
- Pause game
- Restart game
- Timer displaying at the side

## Requirements
- MRTK3
- [Newtonsoft](https://www.newtonsoft.com/json)
